## Usage

### Project Initialization
Command options:

| Options | Description | Required |
|---------|-------------|----------|
| -m      | module name | true     |
| -n      | module name | true     |
| -f      | in folder   | false    |

example:
```
gogen init -m <module-name> -n <project-name> [-f]
```
or
```
gogen init --module-name <module-name> --project-name <project-name> [--in-folder]
```
